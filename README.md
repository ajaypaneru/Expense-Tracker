# Expense Tracker

A simple web-based Expense Tracker application that allows users to record, manage, and track daily expenses efficiently.

## 🚀 Features
- Add new expenses
- Delete expenses
- View total spending
- Organized frontend and backend structure

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (Backend)
- Express.js

## 📁 Project Structure
- frontend/ → UI related files
- backend/ → Server-side logic
